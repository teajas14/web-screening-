# 🎬 Movie Info Extractor

## Overview
This project automates the process of **screening your laptop screen** to detect movie details such as:
- 🎥 Movie Name  
- ⭐ Rating  
- ⏱️ Duration  

The extracted information is then stored neatly into an **Excel sheet** for easy access, tracking, and analysis.

---

## ✨ Features
- Captures the screen of your laptop in real-time.  
- Uses OCR (Optical Character Recognition) and text parsing to identify movie details.  
- Automatically organizes the data into an Excel file.  
- Lightweight and easy to run on any system.  

---

## 🛠️ Tech Stack
- **Python** (core logic)  
- **OpenCV** (screen capture & image processing)  
- **Pytesseract** (OCR for text extraction)  
- **Pandas** (Excel sheet handling)  
- **xlrd / openpyxl** (Excel file support)
